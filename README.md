# WildWings Premium Hotel

WildWings Premium Hotel is a modern luxury hotel website designed with a premium user experience and AI-powered assistance. The platform provides visitors with an elegant interface to explore rooms, dining services, facilities, and hotel information.

## Features

- Premium luxury hotel UI
- AI chatbot assistant
- Responsive modern design
- Rooms showcase
- Dining section
- Facilities section
- Manager login page
- Smooth navigation and animations
- Fast performance using Vite

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
